# Dashboard Navigation Guide

Learn how to access key insights and tools within the Adsparkx dashboard.

## 1. Navigation Sidebar
- **Overview**: Real-time traffic, success rate, and general system health.
- **Analytics**: Historical request patterns, query durations, and active connections.
- **Billing**: Plan details, payment methods, past invoices, and usage statistics.

## 2. CSV Report Export
You can export tabular usage history at any time. Go to Analytics, click the calendar filter to set a date range, and click the 'Export CSV' button on the top-right corner.
