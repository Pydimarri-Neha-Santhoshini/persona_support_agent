# Security Best Practices

Maximize the protection of your workspace credentials and client data.

## 1. Multi-Factor Authentication (MFA)
Enable MFA on all administrative accounts. Navigate to Settings > Security > Multi-Factor Authentication and scan the generated QR code with an authenticator application.

## 2. API Key Rotation
API keys should be rotated every 90 days. Deactivate old keys only after verifying that the newly generated keys are successfully deployed in your production cluster.
